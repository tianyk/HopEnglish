# HopEnglish

**一款面向 2-5 岁幼儿的沉浸式“因果反馈”英语启蒙工具。**

## 核心理念

*   **直觉交互**：无需学习，点一点就会玩。
*   **零翻译**：建立“图像=声音”的直接联系，不依赖中文解释。
*   **因果一致**：每一次点击都有即时、愉悦的反馈。

## 学习主题 (MVP)

*   🦁 动物世界
*   🍎 美味食物
*   🚗 交通工具
*   🏃 动作与状态
*   🏠 居家生活
*   🎵 乐器与声音
*   🌤️ 天气与自然

## 技术栈

*   Flutter 3.27.4
*   Dart 3.6.2

## 系统要求

### iOS
*   **最低版本**: iOS 15.0
*   **支持设备**: iPhone、iPad
*   **架构**: arm64（真机）、x86_64（模拟器）

*   macOS（用于 iOS 开发）
