## ❌ 必须完成项（App Store Connect）

### 1. App Store Connect 账户设置

- [ ] **Apple Developer 账户**: 确保已注册并支付年费（$99/年）
- [ ] **App Store Connect 访问权限**: 确认可以登录 https://appstoreconnect.apple.com
- [ ] **Bundle ID 注册**: 在 Apple Developer 后台确认 `cc.kekek.hopenglish` 已注册

### 2. 应用信息（App Store Connect）

#### 2.1 基本信息
- [ ] **应用名称**: `HopEnglish`（最多 30 个字符，可与主屏幕名称不同）
- [ ] **副标题**: 建议 "幼儿英语启蒙" 或 "儿童英语学习"（最多 30 个字符）
- [ ] **类别**: 
  - 主要类别：`教育` (Education)
  - 次要类别：`儿童` (Kids) 或 `游戏` (Games)
- [ ] **年龄分级**: 选择 `4+`（适合 2-5 岁幼儿）

#### 2.2 应用描述
- [ ] **应用描述**（简体中文，最多 4000 字符）:
  ```
  HopEnglish 是一款专为 2-5 岁幼儿设计的沉浸式英语启蒙应用。
  
  核心特色：
  • 直觉交互：无需学习，点一点就会玩
  • 零翻译：建立"图像=声音"的直接联系
  • 因果一致：每一次点击都有即时、愉悦的反馈
  
  学习主题：
  🦁 动物世界 🍎 美味食物 🚗 交通工具 🏃 动作状态
  🏠 居家生活 🎵 乐器声音 🌤️ 天气自然
  
  让孩子在游戏中自然习得英语，无需翻译，无需识字。
  ```

- [ ] **关键词**: 建议 "英语启蒙,幼儿英语,儿童英语,英语学习,早教"（最多 100 个字符，用逗号分隔）
- [ ] **宣传文本**（可选，最多 170 个字符）: 用于 App Store 搜索结果展示

#### 2.3 应用截图
- [ ] **iPhone 6.7" 截图**（必需）: 至少 3 张，最多 10 张
  - 尺寸：1290 x 2796 像素
  - 建议：首页、学习页、庆祝页各一张
- [ ] **iPhone 6.5" 截图**（可选）: 1242 x 2688 像素
- [ ] **iPad Pro 12.9" 截图**（如果支持 iPad）: 2048 x 2732 像素

**截图要求**:
- 必须使用真实的应用截图（不能是设计稿）
- 不能包含占位符文字（如 "Lorem ipsum"）
- 不能包含价格信息
- 不能包含其他应用的图标或名称
- 第一张截图最重要（用户第一眼看到）

#### 2.4 应用预览视频（可选但推荐）
- [ ] **预览视频**: 15-30 秒，展示核心功能
- [ ] 格式：MOV 或 MP4，H.264 编码

### 3. 隐私与合规

#### 3.1 隐私政策
- [ ] **是否需要隐私政策**: 
  - ✅ 当前应用**不收集**任何用户数据
  - ✅ 数据仅存储在本地设备
  - ⚠️ **建议**: 即使不收集数据，也提供简单的隐私政策说明

**建议的隐私政策内容**:
```
HopEnglish 隐私政策

数据收集：
本应用不收集、不传输任何用户个人信息。所有学习进度数据仅存储在您的设备本地，不会上传到任何服务器。

数据存储：
学习进度数据（如查看次数、播放次数）存储在应用本地数据库（SQLite）中，仅用于优化学习体验。

儿童隐私：
本应用专为 2-5 岁幼儿设计，严格遵守儿童隐私保护规定，不收集任何儿童个人信息。

联系方式：
如有隐私相关问题，请联系：[你的邮箱]
```

- [ ] **隐私政策 URL**: 需要托管在可公开访问的网页上
  - 选项 1: 创建 GitHub Pages 页面
  - 选项 2: 使用自己的网站
  - 选项 3: 使用隐私政策生成工具（如 [Privacy Policy Generator](https://www.privacypolicygenerator.info/)）

#### 3.2 App 隐私详情（App Store Connect）
- [ ] **数据收集声明**: 
  - 选择 "不收集任何数据" 或 "仅收集不关联到用户的数据"
  - 如果选择后者，需要详细说明收集的数据类型和用途

#### 3.3 年龄分级问卷
- [ ] 完成 App Store Connect 中的年龄分级问卷
- [ ] 根据应用内容选择合适的分级（建议 `4+`）

### 4. 应用内购买（如果适用）

**当前状态**: 应用为免费应用，无内购

- [ ] 确认应用定价：`免费`
- [ ] 如果未来需要添加内购，需要：
  - [ ] 在 App Store Connect 中配置内购项目
  - [ ] 在代码中实现内购逻辑
  - [ ] 提供内购项目的描述和截图

### 5. 测试与审核

#### 5.1 TestFlight 测试（推荐）
- [ ] **内部测试**: 邀请团队成员测试（最多 100 人）
- [ ] **外部测试**: 邀请外部测试用户（最多 10,000 人）
- [ ] 测试周期：建议至少 1-2 周，确保应用稳定

#### 5.2 审核信息
- [ ] **审核备注**（可选）: 向审核员说明应用的核心功能和特点
  ```
  应用说明：
  HopEnglish 是一款专为 2-5 岁幼儿设计的英语启蒙应用。
  
  核心功能：
  1. 通过图像和声音的直接关联，帮助幼儿学习英语单词
  2. 所有内容均为本地资源，无需网络连接
  3. 学习进度数据仅存储在设备本地，不收集任何用户信息
  
  测试账号：无需账号，可直接使用
  
  如有问题，请联系：[你的邮箱]
  ```

- [ ] **演示账号**（如果应用需要登录）: 当前应用无需登录，跳过此项

### 6. 构建与上传

#### 6.1 Xcode 配置
- [ ] **签名证书**: 在 Xcode 中配置 Distribution 证书
- [ ] **Provisioning Profile**: 确保使用 App Store Distribution Profile
- [ ] **构建配置**: 使用 Release 配置

#### 6.2 构建应用
- [ ] **Archive**: 在 Xcode 中选择 Product > Archive
- [ ] **验证**: 在 Organizer 中验证 Archive（Validate App）
- [ ] **上传**: 上传到 App Store Connect

**或使用 Flutter 命令**:
```bash
# 构建 iOS Release 版本
flutter build ios --release

# 然后在 Xcode 中 Archive 和上传
```

#### 6.3 版本信息
- [ ] **构建版本**: 在 App Store Connect 中选择上传的构建版本
- [ ] **版本号**: 确保与 pubspec.yaml 中的版本号一致（当前：1.0.0）
- [ ] **构建号**: 确保每次上传都递增（当前：1）

### 7. 提交审核

- [ ] **最终检查**: 确认所有必填项已完成
- [ ] **提交审核**: 在 App Store Connect 中点击"提交以供审核"
- [ ] **等待审核**: 通常 24-48 小时（首次上架可能更长）

## 📝 注意事项

1. **首次上架**: 审核时间可能较长（3-7 天），请耐心等待
2. **拒绝处理**: 如果审核被拒，仔细阅读拒绝原因，按要求修改后重新提交
3. **版本更新**: 后续版本更新流程相同，但审核时间通常更短
4. **关键词优化**: 定期优化关键词，提高应用在 App Store 中的搜索排名
5. **用户反馈**: 关注 App Store Connect 中的用户评价和反馈，及时改进
