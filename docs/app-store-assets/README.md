# App Store 上架资源

本目录存放 App Store 上架所需的所有资源文件。

## 目录结构

```
app-store-assets/
├── screenshots/          # 应用截图
│   └── iphone-16-pro/   # iPhone 16 Pro 截图 (1290 x 2796)
└── preview-video/        # 应用预览视频（可选）
```

## 截图要求

### iPhone 16 Pro（必需）
- **尺寸**: 1290 x 2796 像素（或 1320 x 2868 像素）
- **数量**: 至少 3 张，最多 10 张
- **建议内容**: 首页、学习页、庆祝页各一张
- **存放路径**: `docs/app-store-assets/screenshots/iphone-16-pro/`

## 截图规范

- ✅ 必须使用真实的应用截图（不能是设计稿）
- ✅ 不能包含占位符文字（如 "Lorem ipsum"）
- ✅ 不能包含价格信息
- ✅ 不能包含其他应用的图标或名称
- ✅ 第一张截图最重要（用户第一眼看到）

## 预览视频（可选）

- **时长**: 15-30 秒
- **格式**: MOV 或 MP4，H.264 编码
- **内容**: 展示核心功能和交互体验

## 文件命名建议

### 截图命名
```
screenshot-01-homepage.png
screenshot-02-learning.png
screenshot-03-celebration.png
```

### 预览视频命名
```
preview-video.mp4
```
