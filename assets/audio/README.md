# 音频资源

## 目录结构

```
audio/
└── celebrations/     # 庆祝表扬语音
    ├── great.wav
    ├── well_done.wav
    ├── good_job.wav
    ├── awesome.wav
    └── yay.wav
```

## 录音要求

| 要求 | 说明 |
|------|------|
| 时长 | 1-2 秒 |
| 语速 | 稍慢，清晰 |
| 语调 | 欢快、积极、夸张一点 |
| 声音 | 温暖友好（女声/童声效果更好） |
| 格式 | WAV 或 MP3，44.1kHz |

## TTS 生成工具

推荐使用 Google AI Studio 生成语音：

🔗 https://aistudio.google.com/generate-speech?hl=zh-cn
